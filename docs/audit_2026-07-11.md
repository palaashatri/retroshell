## Honest limitations (still true)

1. **Docker-on-mac compositor** needs DRI3; falls back to labwc. Full DE works. Pi/native Linux is the compositor GPU path.
2. **HDR pixels** require HDR-capable GPU/display; nested path correctly reports `hdr_supported=false`.
3. **AT-SPI registry Embed** may fail if registry not fully up; objects are still exported on the a11y bus.
4. **Orca-perfect a11y**, universal global menus for all external apps, and HiDPI scale tree remain incomplete polish.
