# RFC-0009-Dock.md

Status: Accepted
Version: 1.0

## Abstract

Dock provides application launching, switching, status, and trash. It is always available.

## Layout

Applications, running apps, folders, trash.

## Application States

Not Running, Running, Focused, Attention Required.

## Actions

Launch, Quit, Show Windows, Reveal in Finder.

## Position

Default bottom; left and right optional; magnification optional.

## Trash

Dock owns trash presentation; Finder owns trash contents.
