# RetroShell

RetroShell is a modern desktop environment inspired by Classic Mac OS, NeXTSTEP, and BeOS.

This repository contains the architecture, RFCs, subsystem specs, and agent instructions needed for a fleet of AI coding agents to implement the project in a coordinated way.

Primary principles:
- Desktop-first
- Global menu bar
- Native RetroKit widgets
- Application bundles as `.app`
- No user-facing package manager
- High visual consistency
- Dark mode and accessibility from day one
